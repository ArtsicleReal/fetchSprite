# Fetch Sprite

fs.drawSprite(context, image, spritesheetX, spritesheetY, spritesheetWidth, spritesheetHeight, x, y, width, height);

```javascript
fs.drawSprite(bgContext, images[0], 0+64, 0, 64, 64, 0, 0, 32, 32);
```
